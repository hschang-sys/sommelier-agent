# 🎬 영상 소믈리에 에이전트
### AI가 나를 알아갈수록 정확해지는 개인화 영상 추천 에이전트

이 에이전트는 OTT 플랫폼에서 "뭐 볼까 고르다 20분이 지나는" 문제를 해결하기 위해 만들어졌습니다.
Firecrawl MCP로 IMDB·Letterboxd 실시간 데이터를 수집하고, Memory MCP로 사용자 취향을 영구 저장하여
대화할수록 점점 정확해지는 개인화 추천 에이전트입니다.

## 사용 MCP
- Firecrawl MCP: 영화·드라마 정보 실시간 수집 (IMDB, Letterboxd, 로튼토마토)
- Memory MCP: 사용자 취향·시청 이력·반응 영구 저장 및 학습

## 핵심 기능
1. 기분·상황 기반 실시간 콘텐츠 추천
2. 반응 저장 후 다음 추천에 자동 반영 (학습 구조)
3. Hook 기반 RULES.md 강제 로딩 (매 채팅마다 자동 실행)
4. MCP 작업 후 재조회 검증 및 상태 실시간 출력
5. 오류 시 자동 재시도 + 대체 사이트 폴백

## 실행 방법
Claude Code에서 바로 대화하면 됩니다.
예) "오늘 긴장감 있는 거 보고 싶어, 넷플릭스에서 추천해줘"
